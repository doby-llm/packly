# packly2

App meant to help people pack stuff.

## CI/CD

Android compilation, lint, tests, and APK assembly are planned to run only on GitHub-hosted GitHub Actions runners. The repository does not yet contain Android project files, so the active workflow is intentionally deferred until the Gradle project structure exists.

See `docs/ci/github-actions-plan.md` for the CI/CD design, dependency checklist, and future APK artifact retrieval steps.
